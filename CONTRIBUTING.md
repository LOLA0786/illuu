# Contributing to illuu

## Quickstart (Under 30 Minutes)
1. Create a virtual environment:
   ```bash
   python3 -m venv .venv
   ```
2. Bootstrap dev tooling:
   ```bash
   make setup
   ```
3. Run hygiene checks:
   ```bash
   make fmt
   make lint
   make test
   ```

## Engineering Hygiene Workflow
- **Formatting**: `black` and `isort` are enforced.
- **Linting**: `flake8` and `mypy` run in CI and locally.
- **Tests**: `pytest -q`, `python -m compileall .`, and `pipreqs --use-local --diff`.

## CI Alignment
CI runs the same targets as local development:
- `black --check .`
- `isort --check-only .`
- `flake8 .`
- `mypy .`
- `python -m compileall .`
- `pipreqs --use-local --diff requirements.txt .`
- `pytest -q`

## Dependency Manifests
- `requirements.txt`: production dependencies (generated from repo requirements).
- `requirements-dev.txt`: dev-only tools and testing dependencies.

## Audit-Friendly Practices
- Avoid manual edits to compliance artifacts; use automation scripts.
- Preserve evidence and audit logs generated by automated pipelines.
- Use governance enforcement for all external calls.
