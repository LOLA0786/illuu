# Knowledge Vault

Knowledge Vault is the memory and truth layer of the AI OS.

Responsibilities:
- Ingest structured and unstructured data
- Store versioned, auditable knowledge
- Serve permissioned context to AI agents
- Act as the single source of truth

This service is:
- Agent-safe
- Read-first
- API-driven
- Orchestrator-agnostic

No agent executes actions here.
